Welcome!
Chrome Flags Displays a country flag indicating the location of the website you're currently visiting. The extension also provides access to detailed information regarding the website's server, owner, trustworthiness and more.
Getting Started
During visiting a web page you'll notice a flag icon on the address bar. That flag tells you were the server you're visiting is actually hosted. If you click on the flag, a popup menu provides quick links to details about the given server and website, such as:
  - Who owns it (whois) and what the IP address of the given server.
  - Web analytic information from Google Trends, Quantcast, Compete and     Alexa.
  - McAfee Site Advisor.
  - Users opinion on Web Of Trust.

What other extensions we have made?

IE Tab Classic
IE Tab lets you use Internet Explorer inside Chrome. Some websites are still not compatible with standard browsers, so if you need access to those websites, this extension will assist you.
FlashBlock
This extension automatically blocks all the flash contents on any webpage. Each flash element is being replaced with a placeholder that allows you to load only selected elements on a given page. You can also manage a whitelist of allowed websites via a configuration panel.
Drag N Go
With Drag N Go, browsing the web become faster and more intuitive. Highlight any word or a phrase and instantly search for it on one of several provided search engines by simply drag the text up or down, with or without the Control key pressed.
